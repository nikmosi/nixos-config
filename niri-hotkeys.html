Niri — Горячие клавиши

Шпаргалка всех биндов из home-manager/modules/desktop/niri.nix
Mod = Super (клавиша Windows)
🚀 Запуск приложений
Mod+Enter Терминал (kitty)
Mod+D Запуск приложений (rofi)
Mod+C История буфера обмена (cliphist)
Mod+Shift+P Менеджер паролей (rofi-pass)
Mod+X Меню питания (power-menu)
🪟 Управление окнами
Mod+Q Закрыть окно
Mod+Shift+Q Выйти из niri
Mod+Shift+/ Показать оверлей горячих клавиш
🎯 Фокус (Vim: HJKL)
Mod+H / L Фокус колонки влево / вправо
Mod+J / K Фокус окна вниз / вверх
Mod+Ctrl+J / K Фокус окна-или-воркспейса вниз / вверх
Mod+Shift+Tab Фокус на предыдущее окно
Mod+Home / End Фокус на первую / последнюю колонку
📦 Перемещение окон
Mod+Shift+H / L Двинуть колонку влево / вправо
Mod+Shift+J / K Двинуть окно вниз / вверх
Mod+Ctrl+Shift+J Окно вниз или на воркспейс вниз
Mod+Ctrl+Shift+K Окно вверх или на воркспейс вверх
Mod+Ctrl+Home / End Колонку в начало / конец
🖥️ Мониторы
Mod+Alt+H / L Фокус на монитор слева / справа
Mod+Alt+Shift+H / L Колонку на монитор слева / справа
📋 Навигация по воркспейсам
Mod+Page Down / Page Up Воркспейс вниз / вверх
Mod+U / I Воркспейс вниз / вверх
Mod+Tab / Esc Предыдущий воркспейс
↕️ Перемещение воркспейсов
Mod+Shift+Page Down / Up Сдвинуть воркспейс вниз / вверх
Mod+Shift+U / I Сдвинуть воркспейс вниз / вверх
↗️ Колонку на воркспейс
Mod+Ctrl+Page Down / Up На воркспейс вниз / вверх
Mod+Ctrl+U / I На воркспейс вниз / вверх
🖱️ Колесо мыши (cooldown 150мс)
Mod+Wheel ↓ / ↑ Воркспейс вниз / вверх
Mod+Ctrl+Wheel ↓ / ↑ Колонку на воркспейс вниз / вверх
Mod+Wheel → / ← Фокус колонки вправо / влево
Mod+Ctrl+Wheel → / ← Двинуть колонку вправо / влево
🖥️ Воркспейсы DP-1 (левый монитор)
Mod+1 一
Mod+2 二
Mod+3 三
Mod+4 四
Mod+5 五
Mod+Ctrl+1 → 一
Mod+Ctrl+2 → 二
Mod+Ctrl+3 → 三
Mod+Ctrl+4 → 四
Mod+Ctrl+5 → 五
🖥️ Воркспейсы DP-2 (правый монитор)
Mod+6 六
Mod+7 七
Mod+8 八
Mod+9 九
Mod+0 十
Mod+Ctrl+6 → 六
Mod+Ctrl+7 → 七
Mod+Ctrl+8 → 八
Mod+Ctrl+9 → 九
Mod+Ctrl+0 → 十

Mod+Ctrl+R → chatterino
Mod+Ctrl+T → telegram
🔀 Поглощение / изгнание колонок
Mod+Ctrl+[ / ] Поглотить/изгнать окно влево / вправо
Mod+, Поглотить окно в колонку
Mod+. Изгнать окно из колонки
📐 Пресеты размеров
Mod+B Сменить пресет ширины колонки
Mod+Shift+B Пресет ширины колонки (назад)
Mod+Alt+R Сменить пресет высоты окна
Mod+Alt+Shift+R Сбросить высоту окна
⛶ Максимизация / Полный экран
Mod+F Максимизировать колонку
Mod+Shift+F Полный экран окна
Mod+M Растянуть окно до краёв
Mod+Ctrl+F Расширить колонку на доступную ширину
🎯 Центрирование
Mod+Shift+C Центрировать колонку
Mod+Ctrl+Shift+C Центрировать видимые колонки
↔️ Размеры: точная подстройка
Mod+[ / ] Ширина колонки −10% / +10%
Mod+Shift+[ / ] Высота окна −10% / +10%
🪁 Плавающие / тайлинг
Mod+Y Переключить плавающее окно
Mod+Shift+Y Фокус между плавающим/тайлингом
🗂️ Вкладки / Раскладка
Mod+Shift+T Переключить табовый режим колонки
Mod+Space Следующая раскладка
Mod+Shift+Space Предыдущая раскладка
👁️ Обзор (Overview)
Mod+` Переключить обзор
Mod+O Переключить обзор
📸 Скриншоты
Print Скриншот области
Shift+Print Полный скриншот
Mod+Print Загрузить скриншот
Ctrl+Print Сканировать QR-код
Mod+Shift+Print Меню скриншотов
🔊 Громкость (работает при блокировке)
Vol + Громкость +5%
Vol − Громкость −5%
Mute Включить/выключить звук
Mic Mute Включить/выключить микрофон
🎵 Медиа (работает при блокировке)
▶ / ⏸ Play / Pause
⏹ Stop
⏮ / ⏭ Предыдущий / Следующий трек
☀️ Яркость (работает при блокировке)
Bright + Яркость +10%
Bright − Яркость −10%
🔒 Блокировка / Питание
Mod+Ctrl+L Заблокировать экран (swaylock)
Mod+Shift+Ctrl+P Выключить мониторы
🛡️ Ингибитор горячих клавиш
Mod+Shift+Esc Переключить блокировку горячих клавиш
⌨️ Раскладка клавиатуры
Alt+Shift Переключить раскладку (us ↔ ru)
